    Downloads

    All binaries and packages are available from the latest release (current: v0.0.1-alpha.9).

    Linux

    Package Format Notes
    vpnconnect-linux-amd64 Raw binary No install, run directly
    .deb Debian/Ubuntu sudo dpkg -i
    .rpm Red Hat/Fedora/SUSE sudo rpm -i
    .pkg.tar.zst Arch Linux sudo pacman -U
    .apk Alpine sudo apk add --allow-untrusted
    .ipk OpenWrt opkg install

    macOS

    Package Format Notes
    vpnconnect-macos-installer.pkg macOS installer Recommended — installs daemon + CLI
    vpnconnect-macos.app.zip App bundle Drag to Applications
    vpnconnect-darwin-universal Universal binary (arm64 + amd64) Runs natively on Apple Silicon and Intel
    vpnconnect-darwin-arm64 arm64 binary Apple Silicon
    vpnconnect-darwin-amd64 amd64 binary Intel

    Windows

    Package Format Notes
    vpnconnect-windows-installer.exe NSIS installer Recommended
    vpnconnect-windows-amd64.exe Raw binary No install

    Verify checksums

    Each release includes a checksums.txt. Verify with:

    sha256sum --check checksums.txt
